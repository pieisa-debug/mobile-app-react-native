## Installation

### Prerequisites

*   Install Node.js (14.x or higher)
*   Install Xcode (for iOS) or Android Studio (for Android) on your machine
*   Ensure you have a physical Android or iOS device for testing

### Setup

1.  Clone the repository using the following command:

    ```bash
    git clone https://github.com/username/mobile-app-react-native.git
    ```

2.  Install the required dependencies using npm:

    ```bash
    npm install
    ```

3.  Run the following command to link the native dependencies:

    ```bash
    react-native link
    ```

4.  Build and run the app on your device using the following commands:

    ```bash
    npx react-native run-ios
    npx react-native run-android
    ```
